# Periodic checks for The Banker

## Daily Checks

- **End of day (Shanghai timezone):** Generate daily expense summary
  - Check if day has ended (after 23:59 Shanghai time)
  - Compile all expenses from the day
  - Generate categorized summary with RMB/EUR/USD totals
  - Save to tracking system

## Weekly Checks

- **Sunday 10pm Shanghai time:** Generate weekly expense summary
  - Check if it's Sunday 22:00 Shanghai time
  - Compile all expenses from the week (Sunday to Saturday)
  - Generate categorized summary with RMB/EUR/USD totals
  - Include trends and patterns
  - Save to tracking system

## Monthly Checks

- **End of month:** Generate monthly expense summary
  - Check if month has ended (last day, after 23:59 Shanghai time)
  - Compile all expenses from the month
  - Generate categorized summary with RMB/EUR/USD totals
  - Include spending patterns and insights
  - Save to tracking system

## Regular Maintenance

- Check for any pending expenses that need processing
- Verify currency conversion rates are up to date
- Review categorization consistency
